What Is css?

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML.CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.CSS comes through and specifies your document’s style—page layouts, colors, and fonts are all determined with CSS. Think of HTML as the foundation (every house has one), and CSS as the aesthetic choices

The anatomy of CSS is as follows:

  • selector This is the CSS element name at the start of the ruleset. It defines the elements to be styled. p is the selector. But can be changed depending on hwat you want to style.
  • Declaration This is a single rule like color: red;. It specifies which of the element's properties you want to style.
  • property valueThis specifies what you can change like from red to black.

A quick Summary

Remember the simple points below of how css works:

  • The selector points to the HTML element you want to style.
  • The declaration block contains one or more declarations separated by semicolons.
  • Each declaration includes a CSS property name and a value, separated by a colon.
